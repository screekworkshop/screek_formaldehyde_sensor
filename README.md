# screek_formaldehyde_sensor

## About SFA-1

SFA-1 is an interesting **formaldehyde (HCHO) sensor** based on the **SFA30**.
Although the sensor itself is relatively expensive, our testing shows that it
is **very stable** over long-term use.

You are welcome to join us in exploring this unique and practical sensor.

---

## Firmware & Configuration

The `SFA-1` directory contains the ESPHome **YAML configuration files** used for
this project.

You can freely modify these YAML files to customize the firmware and implement
features that fit your own needs.

---

## DIY Kit

We may also provide **fully prepared DIY kits**, including the required hardware,
for users who prefer a quicker and easier setup.
